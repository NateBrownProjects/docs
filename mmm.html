Getting Started

Download

To download MMM-H.I.V.E, open Terminal and paste the following code: 


cd MagicMirror/
cd modules
git clone https://GitHub.com/NateBrownProjects/MMM-Hive.git
cd MMM-Hive

Installation

Install All dependencies by using:


sh setup.sh

Please make sure that you have installed TheHiveProject, and then copy all files from there into the MMM-Hive folder.
