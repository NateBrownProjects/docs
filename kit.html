Getting Started

Download

To download HiveKit, open Terminal and paste the following code one line at a time: 


cd Documents
mkdir hive
cd hive
pip3 install virtualenv
python3 -m venv ./thp
source ./thp/bin/activate
git clone https://GitHub.com/NateBrownProjects/HiveKit.git
cd HiveKit

Installation

Now that we've installed the project into our computer, we now need to install all the dependencies to get HiveKit working.

If that does not work, eg: showing module not found error, please manually install the modules eg: pip install Speech Recognition

Easy Installation

To install, simlpy paste the following command into your Terminal. 

 
pip3 install -r requirements.txt

Once that is all done, please continue to usage.
