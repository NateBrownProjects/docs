Getting Started

Download

To download H.I.V.E, open Terminal and paste the following code: 


cd Documents
mkdir hive
cd hive
git clone https://GitHub.com/NateBrownProjects/TheHiveProject.git

Installation

Now that we've installed the project into our computer, we now need to install all the dependencies to get H.I.V.E working. There are 2 ways to install the dependencies. If the Easy Install doesn't work or produces an error, please use the Manual Install or Contact Us. We do reccomend using Manual Install as Easy Install contains bugs and may not work on your operating System

Easy Installation

To install, simlpy paste the following command into your Terminal. 

 
cd TheHiveProject
sh setup.sh

Manual Installation

If you would like to install H.I.V.E Manually, you will need to copy and paste the following commands one at a time.


  pip3 install figlet


  figlet Installing H.I.V.E -c -k


  sudo pip3 install virtualenv


  virtualenv TheHiveProject


  source TheHiveProject/bin/activate

This part is REALLY important! Please dont continue if your Virtual Environment hasn't loaded. 


  pip3 install SpeechRecognition


  pip3 install pyttsx3


  pip3 install pywhatkit


  pip3 install datetime


  pip3 install wikipedia


  pip3 install pyjokes


  pip3 install requests


  pip3 install pyowm


  pip3 install wolframalpha


  pip3 install figlet


  chmod +x hivec.sh


  source hivec.sh

Once that is all done, please continue to usage.

Usage

New! - hive command

Part of our new update, you can now use new commands to start H.I.V.E! Simply use the command below!

hive

2nd Method

Or you can use:

sh hive.sh

3rd Method

Or simply just: 
python3 hive.py
